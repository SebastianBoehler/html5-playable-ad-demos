# Glow Lab Cleanse Playable Ad

A mobile-first tactile playable ad for a fictional skincare brand. The player drags a sponge to clean soap off a face, then paints on cream until coverage reaches the target.

## Why this demo exists

- Uses a soft lifestyle visual direction, distinct from the runner and coffee demos.
- Demonstrates continuous drag input, progress coverage and visual product transformation.
- Ends with a product CTA: get the routine kit.

## Run locally

Open `index.html` in a browser, or serve the root with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/html5-game-demos/skincare-lab/`.
