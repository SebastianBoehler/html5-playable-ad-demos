# Speaker Defense

A mobile-first tower defense playable ad for a fictional portable speaker.

The player places and upgrades speaker towers to stop waves of noisy enemies before they reach a rooftop party. The product is built into the mechanics: speakers fire sound waves, upgrades improve the bass/range fantasy, and the end screen resolves into a commerce-style CTA.

## Run

Open `index.html` directly or serve the repo root:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/speaker-defense/`.

## Notes

- Framework-free Canvas 2D.
- DOM HUD and overlays for accessible review.
- Generated sprites and effects, no external assets.
- Designed as the flagship "real game" demo in the set.
