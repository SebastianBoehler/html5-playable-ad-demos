# Brew Rush Playable Ad

A mobile-first dexterity playable ad for a fictional cold brew brand. The player drags a cup across the counter to catch only the green `NEXT` ingredient shown on the ticket. Red `BAD` drops always reduce quality.

## Why this demo exists

- Uses a different genre and visual identity from the shoe runner.
- Demonstrates product assembly, motion, urgency, sound cues and clear wrong-action feedback.
- Ends with a product CTA: claim a starter pack.

## Run locally

Open `index.html` in a browser, or serve the root with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/html5-game-demos/brew-rush/`.
